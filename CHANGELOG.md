# Changelog

本项目遵循语义化版本（SemVer）。

## v1.2.2 - 2026-03-22

- 保留科技黑蓝底的大背景不变，继续做暗色 Apple-ish 细节精修。
- 会话列表加入项目备注与中文说明，便于区分 cron / 主会话 / 各类消息来源。
- 监控台整体样式继续优化：卡片、badge、状态点、边框、阴影与层级更细腻。
- 安全侧新增插件白名单、插件基线、夜间显性安全巡检，并完成 Feishu doc/wiki 能力收缩。

## v1.1.0 - 2026-03-21

### 修复
- 调整 Token 卡片文案，明确区分“近 30 天 / 账单口径”和“累计账本（全历史）”
- 修正累计账本摘要中的总 Tokens 展示，避免把“输入+输出”误显示成“累计总量”
- 增加累计输入输出与累计缓存读取拆分，减少口径误解

### 说明
- 本版本是对监控台 Token 统计展示逻辑的修正版本
- 后续将继续排查 3 月 19 日与 3 月 20 日历史数据补算问题

## v1.0.0 - 2026-03-20

### 里程碑
- 监控台进入首个正式可用版本
- 支持桌面端使用，并开始支持移动端 / PWA 形态

### 新增
- 独立 Git 仓库与 GitHub 私有仓库
- 项目目录整理与 README 完善
- 中文化监控台界面
- 长期账本能力接入（后端累计统计）
- 长期账本摘要整合进 Token 卡片
- 成本统一改为人民币显示
- 基础 PWA 能力：manifest、service worker、安装图标位
- 移动端基础适配：viewport / safe-area / 手机访问基础支持
- 支持局域网访问监控台，可通过同一网络下的手机直接打开

### 修复
- 恢复主版面 6 卡布局
- 修复多次迭代中的前端编译问题
- 统一成本显示逻辑，避免美元与人民币混用

### 说明
- 当前版本的移动端支持为第一阶段：
  - 可作为 Web App / PWA 使用
  - 已具备添加到主屏幕的基础能力
  - 后续仍可继续优化手机端交互与版面
