# Changelog

本项目遵循语义化版本（SemVer）。

## v1.0.0 - 2026-03-20

### 里程碑
- 监控台进入首个正式可用版本
- 支持桌面端使用，并开始支持移动端 / PWA 形态

### 新增
- 独立 Git 仓库与 GitHub 私有仓库
- 项目目录整理与 README 完善
- 中文化监控台界面
- 长期账本能力接入（后端累计统计）
- 长期账本摘要整合进 Token 卡片
- 成本统一改为人民币显示
- 基础 PWA 能力：manifest、service worker、安装图标位
- 移动端基础适配：viewport / safe-area / 手机访问基础支持

### 修复
- 恢复主版面 6 卡布局
- 修复多次迭代中的前端编译问题
- 统一成本显示逻辑，避免美元与人民币混用

### 说明
- 当前版本的移动端支持为第一阶段：
  - 可作为 Web App / PWA 使用
  - 已具备添加到主屏幕的基础能力
  - 后续仍可继续优化手机端交互与版面
